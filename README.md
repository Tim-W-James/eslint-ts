# Install

### ESLint + Prettier

```
npm i eslint eslint-config-prettier eslint-plugin-prettier prettier --save-dev
```

### TypeScript

```
npm i typescript @typescript-eslint/eslint-plugin @typescript-eslint/parser --save-dev
```

### Jest

```
npm i jest eslint-plugin-jest --save-dev
```

# ESLint Config

### Install ESLint

```
npm i -g eslint
```

### Create Config

```
eslint --init
```
